Rubrics Template Creation

Faculty can create Rubrics Template for the particular assignment.

Element Creation

Faculty can create Elements of Rubrics Template in which they can define the criteria for grading the assignment based on points or percentage.

Rubrics Template Permission

For using the created Rubrics template, the faculty have to change the state to 'In Use'.

Assignment Creation And Publish

Faculty can create Assignment and have to publish the assignment, so that the student can submit their assignments. All elements of selected template will be automatically assign to the assignment.

Assignment Submission

Once student submit the assignment, the faculty can assign the marks based on points or percentage defined in the rubrics template and then click on Submit button to obtain the total marks.
