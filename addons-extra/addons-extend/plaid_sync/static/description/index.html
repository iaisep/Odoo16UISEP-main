Automatic Plaid Synchronization

Overview

Overview

This module is used for online bank account synchronization via Plaid. It enables user to link bank journals to their online bank accounts for supported banking institutions via Plaid and configure a periodic and automatic synchronization of their bank statements to get bank feeds directly in odoo.

Description

One thing that companies cannot function without nowadays, is Bank Reconciliations, since they have to interact with financial institutions. As much as it's needed and wanted, it is just as tedious. The biggest problem being lack of standardization and manual reconciliations.
This module enables you to do Online bank synchronization via Plaid. It enables users to link bank journals to their online bank accounts, for supported banking institutions via Plaid, and configure a periodic and automatic synchronization of their bank statements.
You will need to create an account with plaid independently and then use its credentials to link odoo to plaid

Exploring the module: (How to)

Exploring the module: (How to)

Once the module is integrated to your system,open the accounting module.

  • Download and Install the app
  • Open the configuration tab from the menu. The configuration tab provides a convenient hub to the different parts of the Accounting module.
  • Select the online sync via Plaid option from the menu.
  • A form appears requiring the details of the bank. Enter the relevant details. Click connect to Plaid button on top
  • Click on settings in the configuration tab.
  • Select the bank (it should be already created)
  • Go to the accounting settings
  • Open Bank journal.
  • Enter in the given fields. Plaid requires these two
    fields.
    • Client ID
    • Secret
  • Press the Connect to Plaid button.
  • You will get a Bank list. Select the bank (associated with journal).
  • Enter your login details and you'll be connected to Plaid.
  • Bank Statement if its connected press the Get Transaction button.
  • Enter relevant details in the input fields (e.g. dates, Accounts (In case of multiple accounts) etc)
  • Press the save button and bank statements.

The reconciliations are done by Plaid. It is very useful as the time spent on bank reconciliation.
